#MrDang 

具体的行业分类参考 [行业分类](Books/指数基金定投指南.md#^194df7)
小白最应该去了解的是同质化最高的行业：
- 银行，券商，保险，地产，基础化工，煤炭，石油化工，电力，建材，钢铁
看书就要看些行业里面的硬核书籍，就是行业从业人员必须要学的书籍，了解这个行业的常识知识

| 行业   | 代表书籍       |
| ---- | ---------- |
| 银行   | 《货币银行学》    |
| 保险   | 《保险学》      |
| 地产   | 《房地产开发与经营》 |
| 基础化工 | 《化工原理》     |
| 煤炭   | 《煤矿开采学》    |
| 石油化工 | 《石油炼制工程》   |
| 钢铁   | 《冶金原理》     |
| 有色   | 《有色金属冶金学》  |
|      |            |
总之就是行业里面最代表的书籍，当然大部分书看完也是基本了解不多，主要就是对行业基础有个简单的认识，不至于对这个专业什么都不知情就去投资
这个就是增加自己的[能力圈](Zettelkasten/格雷厄姆价值投资理念.md#^20768a)

多看多学，最少了解10个行业，才能知道大致的模糊方向，然后挑选一个预期好的行业，这样才算踏入了价值投资的大门。
