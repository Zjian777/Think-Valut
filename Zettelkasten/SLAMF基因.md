#潜在课题
SLAMF基因全称为signaling lymphocyte activation molecule family，直译是信号传导淋巴细胞激活分子家族

## 2026.2.16
- SLAMF8基因主要表达在**Myeloid cell**中，而且肿瘤组和正常组具有显著差异，Fisher检验P<2.2x10^-16
- 结合标记基因模式判断细胞可能属于哪些细胞类型
- 根据 `DotPlot` 的结果，你可以看到每个cluster高表达哪些标记。以下是一些常见模式的参考：

| 标记模式                                                | 可能的细胞类型                        |
| --------------------------------------------------- | ------------------------------ |
| 高表达 `CD163`, `MRC1`, `C1QA`, `C1QB`，低表达 `S100A8/A9` | **M2型巨噬细胞**（TAM）               |
| 高表达 `IL1B`, `TNF`, `CXCL9/10`                       | **M1型巨噬细胞**（促炎）                |
| 高表达 `TREM2`, `SPP1`, `APOE`, `FN1`                  | **肿瘤相关巨噬细胞（TAM）**，尤其是脂质相关或特定亚型 |
| 高表达 `CD14`，可能同时表达 `FCGR3A`                          | **单核细胞**                       |
| 高表达 `S100A8`, `S100A9`, `VCAN`                      | **经典单核细胞**（可能处于未分化状态）          |
| 高表达 `CD1C`, `FCER1A`, `CLEC10A`                     | **cDC2**                       |
| 高表达 `CLEC9A`, `XCR1`, `CADM1`                       | **cDC1**                       |
| 高表达 `LILRA4`, `CLEC4C`, `TCF4`                      | **pDC**                        |
| 高表达 `LAMP3`, `CCR7`, `FSCN1`                        | **成熟/迁移性树突状细胞（mregDC）**        |

| 高表达 `KIT`, `MS4A2`, `CPA3`                          | **肥大细胞**                       |

- 潜在结论：**SLAMF8在食管癌肿瘤微环境中特异性富集于髓系细胞，尤其是TREM2+肿瘤相关巨噬细胞亚群中**。这为进一步研究SLAMF8在该亚群中的功能提供了依据。
- Limitation
  1. 用到的单细胞数量只有两组，一共四个，因为计算机性能不足，跑不出
  2. 细胞注释可能有问题，TAM_TREM2可能不一定准确
- 可能的科研故事：
	1. **发现**：单细胞测序揭示 SLAMF8 在食管癌髓系细胞中特异性上调，尤其富集于 TREM2+ 巨噬细胞。
	2. **特征**：TREM2+ 巨噬细胞呈现免疫抑制特征（高表达 M2 标记、免疫检查点分子），且 SLAMF8+ 的 TREM2+ 巨噬细胞进一步富集与免疫抑制相关的基因（如 IL10, TGFB1, PD-L1）。
	3. **通讯**：细胞通讯分析显示 TREM2+ 巨噬细胞与 T 细胞之间存在强烈的抑制性相互作用，可能通过 SLAMF8 相关信号。
	4. **临床**：TCGA 数据中 TREM2 和 SLAMF8 的高表达与食管癌患者不良预后相关。
	5. **结论**：SLAMF8 可能是 TREM2+ 肿瘤相关巨噬细胞的关键调节分子，有望成为免疫治疗新靶点。
验证思路：
- **体外实验**：
    - 从外周血单核细胞诱导巨噬细胞，用 TREM2 激动剂或过表达 TREM2 建立 TREM2+ 巨噬细胞模型。
    - 敲低或过表达 SLAMF8，检测巨噬细胞的 M1/M2 标记变化、细胞因子分泌（IL-10, TGF-β）、对 T 细胞增殖的抑制能力。
- **体内实验**：
    - 在髓系细胞特异性敲除 SLAMF8 的小鼠中接种食管癌细胞，观察肿瘤生长、免疫浸润（流式细胞术、免疫荧光）。
- **临床样本验证**：
    - 用多重免疫荧光染色食管癌组织切片，共定位 SLAMF8、TREM2、CD68，观察其空间分布与临床分期的关系。
