#经济学 #读书笔记
# 一切的开始

一座岛，三个人，每个人每天需要吃一条鱼才能生存，但是由于生产力落后，每个人每天能且仅能捉到一条鱼，正好满足每日的生存需求。睡觉，捕鱼，吃鱼，睡觉就是他们生活的全部。
但是想要偷懒是人类创新的源动力，因此在某天，三个人中的艾伯想到可以做一个渔网捕鱼， 但是需要花费自己一天的时间，如果想要制作渔网，那么就要挨饿。艾伯决定冒险一把，选择饿一天，幸运的是他成功了。现在艾伯有了一个渔网，这个渔网每天可以帮他捕捉两条鱼。这样他就可以一天抓两天的鱼，如此他就可以一天捕鱼，一天休息。

“工具改变了一切，使经济的出现成为可能”
“努力使有限的资源（每种资源都是有限的）产生最大的效益以尽可能满足人类的需求，这就是经济这一概念最简单的定义。”

# 把你的财富分享出来吧

艾伯将他多出来的鱼，以投资的方式给另外两个人，让他们不用冒险也能做出渔网，等他们做成渔网后，将艾伯投资的鱼加倍汇报给他。如此，艾伯冒险得到的资本，也帮助了这个岛上其他人。
现在岛上每个人都有渔网了，每个人的生产效率都翻倍，每个人都可以一天工作一天休息了。

“富人致富的原因（至少开始时）是他们为他人提供了有价值的东西。艾伯就为那些没有足够储蓄的人提供了储蓄。如果他赢利了，那也是因为他提供的服务对别人是有价值的。”

# 艾伯借鱼的另一条世界线

在书中的世界线中，艾伯为了另外两个人能够做出渔网，将自己多的鱼借出。但如果另外两个人借鱼的目的不是为了做渔网呢？如果是以下的目的：
- 纯粹摆烂消费，我不想努力了，我只要你的鱼
- 其他两个人出现了特殊事故，没办法自己捕鱼了
显然对于第一种目的，艾伯肯定不会借，这是个纯粹亏本的生意，但是第二种目的，艾伯最好还是应该作为一个应急借款借出来。不然岛上只剩一个人了，那就完蛋咯。

*“影响个体贷款人的往往是贷款的财务结果，而不是基本活动的政治象征意义”* 
*“贷给个人或者企业的款项如果无法成功促成必要的创新或者提高产能，就会浪费储蓄的供给，削弱整体经济”* 

# 鱼，鱼，鱼，我们需要更多的鱼

仅仅只有渔网，对这个只有三个人的岛来说，生产力的提升还是太少了，每个人还是需要花一天时间捕鱼。我们需要更好的工具，更加自动化的仪器，更加解放生产力的方式。岛上有个人，贝克，提出可以三个人合资，建造一个大型捕鱼器，这样他们除了一些基本的维修工作，每天都不需要干活了都能得到鱼，鱼，以及更多根本吃不完的鱼。
没有捕鱼的生产压力之后，三个人就有时间去做其他事情了，在业余时间里面，三个人创造了自己的成就。具体见下面的表格

| 名字  |   产业    |
| :-: | :-----: |
| 查理  |   冲浪板   |
| 艾伯  |  服装公司   |
| 贝克  | 独木舟和手推车 |

>我们的消费不能超过产能，我们的借款不能超出存款，至少不能长期这样
>储蓄创造了资本，而资本使生产扩大成为可能

# 货币是怎么产生的

三个人的“豪华”生活吸引了其他岛上还在徒手捕鱼阶段的人们，一时间，一大批人涌入了这个小岛。

*“生产率提高意味着这座岛屿可以养活更多的人，反过来，更多的人又促成了更加多样化的经济。”*

现在岛上不只有渔夫了，多元的经济催生出了多元的职业，现在我们有了制矛师，棚屋建筑工人、木筏制造工人、马车制造工人。

*“岛上的食物和工具生产达到了很高的水平，一部分人不用生产物质资料也能生存”* 

这样，服务业出现了，比如大厨和冲浪教练。
目前在这个岛上还是实行原始的以物换物，但是对于不同的人来说可能需求不一样，无法有双方都满意的物品交换。所以我们需要一个中间物来作为交换的媒介，而鱼又是我们每天都需要的，自然鱼成为了我们中间的交换物，换句话说——货币。
