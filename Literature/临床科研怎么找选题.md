
# 临床研究问题的来源
- 诊断
  1. 病因不明确：病因和发病机制不清
     - 某个疾病有多么多么重要，ABCDE可以解释80%的发病病因，但还有20%的病因无法解释，在CNS里面，有研究显示某一个Biomarker，某一个物质，和这个疾病的发生有关，或者是潜在的靶点，我目前有人群，或者有生物标本，所以我要在这个基础上进一步去验证它，到底如何
  2. 不典型临床表现
     - 相当比例的患者并不具有医生或患者本人所熟知的特定疾病的表现形式
     - Case report，对每个疾病的认识研究都是从病历报告开始的
  3. 早期诊断方法的研究
     - 走向早起、精准、定量、定位的精准诊断
     - 寻找更新、更早、无创、方便、经济的诊断方法（比如ABI--PAD）
  4. 诊断标准的研究
     - 疾病的划分一般是二元的（有病/无病），但是疾病的发展常常是渐变与突变、量变与质变相结合的过程
     - 高血压的标准？妊娠期糖尿病空腹血糖的标准？
  5. 误诊和漏诊问题
     - 横断面研究
- 治疗
  1. 新治疗方法的研究
     - 药物、仪器、装置等等
     - 主要采用RCT评估其安全性和有效性
  2. 临床诊疗指南更新的研究
     - 基本做不了，别想
  3. 临床诊疗指南的实施研究
     - 研究指南和临床实践的差异
     - 如何把已经证实有效的Evidence，把它更好地转化为临床实践
  4. 真实世界研究
     - 持续大量收集在常规临床实践接受治疗的患者信息和进行随访，了解治疗方法在真实世界的治疗效果，不良反应，和影响治疗的因素
  5. 药物流行病学和药物基因组学研究
     - RCT的结果是平均效应，基因组学更加个体化
- 预后
  1. 疾病自然史
     - 生物学发病阶段，亚临床阶段，临床阶段，结局发生阶段
  2. 影响预后的因素
     - 预后因素的寻找是最重要的选题方向
  3. 疾病预后的预测模型
     - 建立新模型
     - 改善新模型：现有的模型不适合目前的现状，是否添加新因素更好预测
